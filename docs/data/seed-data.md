# 种子数据说明

> 版本：v1.1.0 | 更新日期：2026-05-22

## 初始化机制

应用启动时 `Program.cs` 调用 `SeedData.InitializeAsync()`，采用"检查式创建"策略：
- 先查询关键数据是否存在
- 不存在才创建，避免重复插入
- 管理员密码不匹配时自动重置

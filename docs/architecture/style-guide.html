Refresh Theme 01

从“热闹点餐页”改成“清爽、可信、好操作”的餐饮经营系统。

主色从大面积红色收敛为绿色业务色，红色只留给价格和关键动作。顾客端突出菜品和购物车，商家端突出订单效率，管理端突出筛选、表格和状态判断。

主题原则

这版不追求花哨，而是让真实用户少找、少想、少误点。

少色块头部不再大面积红色，页面主背景轻、内容区白，重点按钮才上色。
强层级标题、状态、金额、操作各有固定位置，减少视觉跳动。
低圆角业务系统控件保持 6-8px 圆角，避免过于卡通。
可落地所有静态稿都按当前路由和功能模块组织，便于后续拆成 Vue 组件。

设计变量

主色为“竹绿”，辅以珊瑚橙、订单蓝、优惠金，避免整套系统只有一个色相。
